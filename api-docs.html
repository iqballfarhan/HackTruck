Overview

Welcome to the HackTruck API documentation. HackTruck is a platform that connects truck drivers with users who need cargo transportation services. This API provides endpoints for user authentication, post management, and AI-powered recommendations.

Authentication

Most endpoints require authentication using JWT tokens. To authenticate, include the token in the Authorization header as follows:

Authorization: Bearer <your_token_here>

User Roles

The API has two user roles:

  • user: Regular users who can browse posts and get recommendations
  • driver: Drivers who can create, update, and delete their own posts

Response Format

All responses are returned in JSON format.

Authentication

POST /api/auth/register
Register a new user

Create a new user account with email and password authentication.

Parameter Type Description
email Required String User's email address
password Required String User's password
role Required String User role: "user" or "driver"
name Required String User's full name

Example request:

{
  "email": "john@example.com",
  "password": "secret123",
  "role": "driver",
  "name": "John Doe"
}
Successful response (201 Created):
{
  "user": {
    "id": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
    "email": "john@example.com",
    "role": "driver",
    "name": "John Doe"
  },
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
Error response (400 Bad Request):
{
  "message": "Email already exists"
}

Posts

GET /api/posts Requires Auth
Get all posts with filtering and pagination

Retrieve a list of posts with optional filtering, sorting, and pagination.

Parameter Type Description
page Optional Number Page number for pagination (default: 1)
limit Optional Number Number of posts per page (default: 10)
search Optional String Search term for matching origin or destination
truckType Optional String Filter by truck type (e.g., "box", "pickup", "refrigerated")
sortBy Optional String Field to sort by (default: "createdAt")
order Optional String Sort order: "ASC" or "DESC" (default: "DESC")

GET /api/posts?page=1&limit=10&search=Jakarta&truckType=box&sortBy=price&order=ASC
